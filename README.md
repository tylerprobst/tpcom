# tpcom

Personal site for [Tyler Probst](https://github.com/tylerprobst), built with Next.js.

## Development

```bash
git checkout feature/nextjs-village-hero
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the village hero.
